为复杂查询而生的超轻量 SQL 框架

Rabbit-SQL 是比 ORM 更轻的动态 SQL 管理框架，对复杂查询说 Yes，对 ORM 限制说 No！

开始使用 Github

国产化

核心代码完全自主实现，非套壳，对数据安全、审计要求极高的政务项目，可以通过静态 SQL 管理 + 参数化执行，实现最安全可控的数据访问。

独创 XQL 文件，天然适配国产生态，兼容主流国产数据库，如达梦、人大金仓、腾讯 TBase、TD-SQL 等。

极简敏捷

约定大于配置，默认情况下几乎零配置，不会用 ORM？没关系，写 SQL 就能上项目，复杂度由你掌控，更快完成开发任务。

有史以来最简单的动态 SQL，拒绝代码拼接，让 SQL 写法回归它该有的简洁和自由。

开发赋能

支持主流开源框架 Spring Boot 自动配置，最少3行代码可完成一次查询。

插件：支持主流开发工具 IDEA，让 DBA 轻松参与项目，开发、测试流程化，提高沟通效率，降低心智负担，所见即所得。

快照

动态SQL

不写 XML，不绑注解，SQL 自由表达，掌控你的 SQL，而不是被框架绑定。

自创的 XQL 动态 SQL 注释扩展脚本语法，还让动态 SQL 测试不再是一件头疼的事。

接口映射

除了注入核心接口 Baki 之外，还有可选的 XQL 映射接口，支持多种方式来提高灵活性，以满足不同的风格需求。

多实例

多数据源多 Baki 实例配置照样轻松，默认自动配置 Baki 对应默认的 数据源，更多的数据源配置在 secondaries 节点下即可，无需编写任何配置类。
